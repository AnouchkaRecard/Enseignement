Cours : Les vecteurs et les coordonnées de vecteurs

Introduction

Un vecteur est une entité mathématique qui a à la fois une direction et une intensité (ou norme). Contrairement à un scalaire qui est simplement un nombre, un vecteur est représenté par une flèche dans un espace.

Définitions importantes

Vecteur
Un vecteur est un objet qui possède une direction et une norme. Il peut être représenté par une flèche dans un espace géométrique. Un vecteur est défini par ses composantes dans un repère donné.

Coordonnées d’un vecteur
Les coordonnées d'un vecteur correspondent à ses composantes dans un système de coordonnées donné, généralement dans un repère orthonormal. Pour un vecteur ( \vec{v} ) dans ( \mathbb{R}^2 ), ses coordonnées sont de la forme ( \vec{v} = (x, y) ), où ( x ) et ( y ) sont les projections du vecteur sur les axes ( x ) et ( y ).

Représentation des vecteurs

Représentation graphique

Un vecteur peut être représenté graphiquement par une flèche dans le plan ou dans l’espace. La longueur de la flèche correspond à la norme du vecteur et la direction de la flèche correspond à sa direction.

Représentation algébrique

Un vecteur ( \vec{v} ) peut aussi être représenté par un couple ou un triplet de nombres dans un repère donné. Par exemple, dans ( \mathbb{R}^2 ), un vecteur ( \vec{v} = (x, y) ) est déterminé par ses coordonnées ( (x, y) ).

Opérations sur les vecteurs

Addition de vecteurs

Si ( \vec{v} = (x_1, y_1) ) et ( \vec{w} = (x_2, y_2) ), l'addition de ces vecteurs donne : [ \vec{v} + \vec{w} = (x_1 + x_2, y_1 + y_2) ]

Multiplication d’un vecteur par un scalaire

Si ( \vec{v} = (x, y) ) et ( k ) est un scalaire, alors la multiplication de ( \vec{v} ) par ( k ) est : [ k \cdot \vec{v} = (k \cdot x, k \cdot y) ]

Conclusion

Les vecteurs sont essentiels dans de nombreuses branches des mathématiques et de la physique. Ils permettent de modéliser des phénomènes tels que les déplacements, les forces, et bien plus encore. Leur utilisation dans un repère est facilitée par leurs coordonnées et les opérations qui peuvent être effectuées sur eux, comme l'addition et la multiplication par un scalaire.
